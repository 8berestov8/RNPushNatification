# React Native Push Notificatoin

## Installation

```sh
git clone https://github.com/8berestov8/RNPushNatification.git
```

Install the dependencies and devDependencies.

```sh
cd RNPushNatification
yarn install
```

For development environments...

```sh
yarn start
yarn android
yarn ios
```
